How to submit a job

General submission

1. Example data can be chosen or downloaded from the drop-down selections.Three example data can be downloaded to check the input format, or directly load the example expression matrix.

Note: If the user directly loads the example data and leave all settings as default, it will be directed to the example result page without waiting for the program running. Any other changes will be regarded as a new job.

2. Upload your expression file(s) by selecting or dragging the file from local to the drop-zone.

Note: The maximum number of upload files is 3, typically designed for your 10X genomics raw gene-barcodes matrix files. Note that do not unzip your files, otherwise, our server will consider you uploading a text format gene expression file and will try to display the preview.

3. Choose the species. Either Human or Mouse.

Note 1: only selecting species after your file is completely uploaded.

Note 2: choosing the wrong species may lead to low gene matching rate or occur errors.

4. Advanced options. See below for details.

5. An email is not required to submit the job; however, we strongly suggest you provide your email to receive a notification once the job is completed.

6. Submit the job once everything is ready.

7. Check our example result page.



Advanced options


1. Quick selecting a recommended parameter, either a set of parameters with less running time or using default parameters but longer running time.

2. Control the output biclusters from QUBIC. Details can be found here.

3. The upstream promoter region is open to users to decide the region for motif finding. Wider range may cause require longer computational time. Default: 1000 bp.

4.Resolution for clustering in Seurat (form 0-1). A larger number will generate more clusters and a smaller number will generate fewer clusters. Default: 0.8.

5. Enable imputation step using DrImpute with all default parameters in the pre-processing step. Default: disabled.

6. Enable dual strategy for QUBIC2 biclustering optimization. Time-consuming. Not recommended for 10X data. Default: disabled.

7. Provide cell labels for CTSR identification. This file contains two columns: cell names and cell labels. Regulons will be predicted based on the provided cell labels.

8. Upload a gene module file for additional analysis. The uploaded file can include single/multiple columns, and each column represents a gene module.
