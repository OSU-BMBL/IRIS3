How to submit a job

General submission

1. Example data can be chosen or downloaded from the drop-down selections.

2. Upload your expression file by selecting or dragging the file from local to the dropzone box.

3. Choose the species. Either Human or Mouse. Note that, choosimh the wrong species may lead to low gene matching rate or occur errors. In some cases, you may need to wait and re-select the species untill your file finishes upload.

4. An email is not required to submit the job; however, we strongly suggest you provide your email in order to receive notice once the job is completed.

5. A preview of your data will show if you upload a text format file.

6. The maximum number of upload files is 3, typically designed for your 10X genomics raw gene-barcodes matrix files. Note that do not unzip your files, otherwise our server will consider you uploading a text format gene expression file and will try to display the preview.

Note: If the user directly load the example data and leaving all settings as default, it will be directed to the example result page without waiting for the program running.

Users who download the example data and upload for a job run will be regarded as a new job submission.

7. The "Example output" will show you the result page of the Mouse lung epithelial (Zeisel et al. 2015) dataset, without running the program.


Advanced options


1. Quick selecting recommended parameter, either a set of parameters with less running time or using default parameters but longer running time.

2. Select for imputation process (recommended for 10X Genomics data; very time-consuming).

3. Control the output biclusters from QUBIC. Details can be found here

4. Users can provide a cell label file with a "cell names" column matching the names in the expression file (optional), and the second column indicating the ground truth cell labels. Cell labels will be used in two ways: (i) assess the cell type prediction results from Seurat, and (ii) assign CTSRs. If no cell label file is uploaded, the pipeline will automatically use the predicted clusters from Seurat for the following regulon predictions.

5. Users can choose to use either the prediction cell cluster result from Seurat or the ground truth cell types uploaded above for the following CTSR prediction.

6. The "upload gene module" function allows users provide their own gene lists and get them analyzed by IRIS3. The uploaded file can includes single/multiple columns, and each column represents a gene module.

7. The upstream promoter region is open to users to deside the region for motif finding. The default value is 1000 bp. Wider range may cause require longer computational time.
