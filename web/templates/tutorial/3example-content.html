Example result illustration

We applied IRIS3 on a single-cell RNA-seq dataset consisting of 3005 cells from Mouse lung epithelial from Zeisel, et al.. Default parameters (k=20, o=500, f=0.7) were used for QUBIC2 biclustering; Dual strategy was enabled, imputation was disabled. We use the ground-truth labels provided by the paper; regulons were inferenced using DMINDA and MEME based on the clustering result.

This tutorial will guide you through the output page of IRIS3 in details.

Cell clustering

The clustering result includes two parts: an interactive UMAP and a table. A floating window will show up when move your cursor on cells, indicating the belonging cell cluster and cell name. A table showing the four Rand indexes for the predicted cell type evaluations. Four index scores are used to evaluate the closeness of the predicted cell type to the benchmark.

Two high resolution UMAP plots show the benchmark cell label and predicted cell clusters, respectively.

The top 100 marker genes in each cluster generated from the differential expression analysis using Seurat.

A trajectory plot is generated from Slingshot integrated in IRIS3.

A bar plot showing the silhouette score that measures how similar each cell is to its type compared to other clusters.

A Sankey plot showing the convergence and divergence of a two cell groups (ground truth versus Seurat prediction). The width of the arrows shown proportionally to the flow quantity. Note: if ground-truth cell labels are not provided by the user, the Score table and Sankey plot will be omitted.

Regulon heatmap

A table summarizes the clusters shows the overall cell number and regulon number in each cluster. Users’s cell label will be shown in the table, otherwise, only index number will be given to represent cell clusters. A cell-gene-regulon heatmap will be displayed for each cell type by clicking on the CT# tab. See figure details below.

The heatmap, empowered by Clustergrammer, showcases the expression pattern of genes from the top ten regulons in the corresponding cell type. To explore details about Clustergrammer, please read the Clustergrammer documentation.

Both gene compositions of regulons and their expression values across different cell types can be intuitively displayed in such a heatmap. Regulons are ranked in increasing order of the empirical p-values of RSS as described above, and a regulon is named as CTn-Rm with n representing the index of cell type and m represents the regulon rank. Due to the space limitation, only the top ten regulons and their corresponding genes are showcased in the heatmap, and the component genes of each regulon are indicated as green rectangles. The heatmap records the log-transformed expression level of each top-ten-regulon-covered gene across all cells.

Cell names, user-provided cell type labels (if submitted), and cell types labels predicted by Seurat are shown on the heatmap. This heatmap can also be sorted by gene and cell by double-clicking on the appropriate area on the image. Conveniently, a series of gene enrichment tests can be directly performed on the heatmap using the integrated Enrichr function in the Clustergrammer framework.

The complete regulon-gene list can be downloaded by clicking on the download buttons above the heatmap (either in gene symbol or Ensembl ID), and users can switch to regulon results in other cell types by clicking on the corresponding labels.

Regulon details

  1. Each regulon is named by the cell cluster index and regulon number. A regulon is a cell-type-specific regulon (CTSR) if the p-value of regulon specificity score (RSS) is less than 0.05.
  2. Stars indicate marker genes, that is, the differential expressed gene, identified in each cluster using Seurat.
  3. Gene symbols and link to the GeneCards (Human) or the Mouse Genome Informatics (MGI) website.
  4. Corresponding gene Ensembl ID and link to the website.
  5. Gene expression UMAP and comparison to the cell types.
  6. The corresponding TF with a corresponding link to HOCOMOCO database.
  7. Detailed motif finding results from including positions, sequences, position weight matrix, etc.
  8. Motif details linking to the TOMTOM database.
  9. A Clustergrammer heatmap showing the expression value of all genes Of this regulon and cell type.
  10. Enrichr link to the enrichment analysis of this regulon.
  11. Bulk ATAC peak enrichment test. See details here.
  12. Coverage of regulon genes and the topologically associated regions. See details here.
  13. The regulon activity UMAP and comparison to the cell types.
  14. Trajectory analysis colored by cell type and regulon activity, respectively.
