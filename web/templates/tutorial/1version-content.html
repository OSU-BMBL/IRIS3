News

5/2/2020 v1.2.1

  • The manuscript IRIS3: Integrated Cell-type-specific Regulon Inference Server from Single-cell RNA-Seq has been officially accepted by Nucleic Acids Research!

4/18/2020 v1.2.1

    Changed:
  • We updated our Tutorial and FAQ with more detailed methods and typo fixed.
  • We updated p-value calculation methods in assigning gene modules.
  • We updated RSS scatter plots for all regulons.
  • We removed p-values for motifs and highlighted CTSR in result page.
  • We added option to allow select p-value threshold for regulon specificity score.

4/3/2020 v1.2.0

    Changed:
  • We updated our Tutorial and FAQ, the new version contains more detailed steps for new designed pages.
  • We updated the download files list for interactive UMAP plot.
  • We added the job running status and cell clustering results are available to preview on the submission waiting page, we also added a “Terminate your job” button for users to stop the current job, in the event that the clustering result does not make sense or to save time for parameter tuning.
  • We improved the layout of the results page by splitting it into four sections: (i) cell clustering page, including preprocessing results, interactive UMAP, differentially expressed genes, trajectory plot, and clustering evaluations; (ii) the cell-gene-regulon Clustergrammer heatmap page, (iii) regulon details per cell type page, and (iv) job information page. The heatmap is now closed by default, and users can choose to turn it on by clicking on the corresponding cell type buttons.
  • We changed “marker genes” into “differentially expressed genes” (DEGs for short) throughout the webserver.
  • We enabled users to choose different number of PCs, number of highly variable genes, and clustering resolution for Seurat cell clustering.
  • In calculating the regulon specificity score(RSS), the p-value for RSS in each cell cluster is adjusted by multiplying the number of the TFs (annotated by HOCOMOCO) that are also identified as differentially expressed genes by Seurat.
  • We added 'zip','gzip','tar.gz','7zip' as supported gene expression input format.
    Fixed:
  • Fixed email sometimes failed to send.
  • Fixed some test jobs not available to show.
